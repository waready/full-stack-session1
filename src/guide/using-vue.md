# Instalación de herramientas 

## Git
Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente

[Git directo](https://github.com/git-for-windows/git/releases/download/v2.38.1.windows.1/Git-2.38.1-64-bit.exe)
[Git](https://git-scm.com/downloads)

## Node js

Necesitamos tener node instalado sugerimos la versión 14.17.6 para no tener conflictos con las librerías que vamos a utilizar.

[Node](https://www.npackd.org/p/org.nodejs.NodeJS64/14.17.6)

una vez finalizada la instalación verificamos en la consola:

``` sh 
node -v
v14.17.6

npm -v
6.14.15
```

## XAMPP
XAMPP es una distribución de Apache fácil de instalar que contiene MariaDB, PHP y Perl. Simplemente descarga y ejecuta el instalador. ¡Es así de fácil!
Usaremos la versión 7.4.30 / PHP 7.4.30 de PHP

[Xampp](https://www.apachefriends.org/es/download.html)


## Postman

Postman es una plataforma de API para que los desarrolladores diseñen, construyan, prueben e iteren sus API. 

[Postman](https://www.postman.com/downloads/)
